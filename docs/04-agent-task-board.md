# AI 聊天式电商项目 Agent 开发任务看板

## 文档说明

本文件用于统一维护当前 AI 聊天式电商项目的多 Agent 并行开发进度。

### 当前协作模式
- 采用 4 Agent 并行开发：
  1. 架构与契约 Agent
  2. 后端业务 Agent
  3. AI 模块 Agent
  4. 前端 Agent

### 当前项目冻结约束
- 商品表名：`product`
- 商品字段固定：
  - `id`
  - `category`
  - `title`
  - `price`
  - `sales`
  - `image_url`
  - `shop_name`
  - `embedding`
- 当前只支持 **SPU**
- 当前不支持：
  - SKU
  - 库存
  - 多规格
  - 复杂订单拆表
- 订单模型：
  - `order_info` 主表
  - `items_json` 存商品快照
- 登录方式：
  - 用户名/邮箱 + 密码
- 鉴权方式：
  - JWT Bearer Token
- 支付方式：
  - 支付宝沙箱 PC 网页跳转支付
- 前端页面固定：
  - `/login`
  - `/register`
  - `/home`
  - `/chat`
  - `/cart`
  - `/orders`
  - `/payment/result`
